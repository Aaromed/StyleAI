# **App Name**: StyleAI

## Core Features:

- Intelligent Cloth Analyzer: An AI-powered tool to detect clothing types, primary colors, and style classifications from uploaded photos.
- Virtual Wardrobe Manager: A comprehensive digital closet to browse and manage your categorized items stored in the cloud.
- Adaptive Outfit Generator: An AI reasoning tool that recommends matching outfits by evaluating weather conditions, color theory, and clothing styles.
- Live Weather Context: Real-time weather data integration from OpenWeather API providing local temperature and precipitation alerts.
- Authentication Hub: Secure sign-up and login management using Firebase Authentication for a personalized experience.
- Unified Style Settings: A centralized profile screen to manage personal details and account preferences.

## Style Guidelines:

- The palette is inspired by High Fashion, using a sophisticated Dark Scheme.
- Primary Color: Royal Orchid (#A66BCC), a vibrant and tech-forward purple that provides a bold pop against the dark backdrop.
- Background Color: Obsidian Midnight (#141116), a heavily desaturated deep violet to maintain elegance and focus on high-quality visuals.
- Accent Color: Periwinkle Mist (#949AFB), used for active states and secondary UI elements to provide an analogous high-contrast contrast.
- The font 'Poppins', a geometric sans-serif, is recommended for all text to achieve a precise, contemporary, and avant-garde fashion magazine look.
- Minimalist, thin-line vector icons to represent weather patterns and clothing categories, keeping the interface airy and uncluttered.
- Smooth card transitions and subtle 'scanning' effects during the clothing upload and analysis phase.
